# How DohDocs Works

Welcome to DohDocs — your shared notebook for job notes, checklists, and anything worth writing down. You don't need to know any special "code" to use it: everything is done through the toolbar at the top of a doc. This guide walks through the editor, the formatting tools, sharing, and live collaboration.

## The basics

- **Create a doc** with the **+** button in the sidebar. Give it a title at the top.
- **Everything auto-saves.** There's no Save button — your changes are written a fraction of a second after you stop typing.
- **Find docs** with the search box. Sort by *Last edited* or *Name*.
- **Organize with folders.** Drag docs into folders in the sidebar; folders can be nested.

## Writing & formatting (the toolbar)

You format text with the ribbon at the top of the editor — no Markdown knowledge needed.

- **Heading** — turn a line into Heading 1–4, or back to Normal text. Headings are styled in small-caps for a clean, technical look.
- **List** — switch the current line(s) to:
  - **Bulleted** list
  - **Numbered** list
  - **Task (checkbox)** list — clickable checkboxes you can tick off
- **Image** (paperclip icon) — attach an image straight into the doc.
- **Archive Done** — sweeps your completed checkbox items into a separated "archived" section at the bottom so your active list stays clean. The tasks aren't deleted, just moved aside.

> Tip: type `- [ ]` at the start of a line and it automatically becomes a checkbox task.

## The "F" formatting ribbon

Click the **F** button to open the special formatting selectors. These are DohDocs' own highlights and tools — select some text first, then click one:

| Button | What it does |
|--------|--------------|
| **P1** | Red highlight — Priority 1 |
| **P2** | Yellow highlight — Priority 2 |
| **P3** | Blue highlight — Priority 3 |
| **Comment** | Styles the text as an italic, quoted aside |
| **Math** | Treats the selected text as a calculation (e.g. `12 * 3 + 6`) and appends the answer (`= 42`) |
| **Clear** | Removes any of the above formatting from the selection |

Use **P1 / P2 / P3** to color-code priorities at a glance, **Comment** for side notes, and **Math** when you want a quick on-the-spot calculation right in your notes.

## Comments (discussion threads)

Beyond the editor, DohDocs has Google-Docs-style **comment threads**:

- Select text and click the **comment** (speech-bubble) button to start a thread anchored to that spot.
- Comments open in a side panel. You can **reply** to build a thread, and **resolve** a thread when it's handled.

## Sharing

Every doc and folder is **private to you by default**. Nobody else sees it until you share it.

**Share a doc** with the share button in the editor toolbar (the connected-dots icon — only the owner sees it). A panel slides in where you can:

- Search for a person or a group and add them.
- Choose their access level:
  - **Edit** — full read and write access.
  - **Comment** — they can read and join comment threads, but can't change the text.
- Remove access at any time.

**Share a whole folder** from the folder's **⋯** menu → **Share folder**. Everything in that folder inherits the access. (A doc's own sharing always overrides whatever the folder grants.)

### The three sidebar views

A toggle at the top of the sidebar switches what you're looking at:

- **Mine** — only docs you own, in your folders.
- **Shared** — docs other people have shared with you, grouped by owner, each with an Edit/Comment badge.
- **All** — your docs and folders, plus a "Shared with me" section.

Folders you own that you've shared with someone show a small share icon in the sidebar.

> If a doc was shared with you as **Comment** only, the editing tools are hidden and a read-only banner appears — but you can still comment.

## Live collaboration

When more than one person has the same doc open:

- A **presence bar** shows who's currently viewing it (and who's actively editing).
- Edits show up **live** — when someone saves, everyone else's view refreshes automatically.

## Sharing & exporting outside DohDocs

- **Copy** a doc and it carries its formatting into Word, Gmail, etc. (it copies both rich text and plain text).
- **Export to PDF** uses your browser's print dialog.

## Quick reference

| I want to… | Do this |
|------------|---------|
| Make a checklist | List → Task (checkbox), or type `- [ ]` |
| Flag a priority | Select text → **F** → P1 / P2 / P3 |
| Do quick math | Select the expression → **F** → Math |
| Add a side note | Select text → **F** → Comment |
| Discuss a passage | Select text → comment button |
| Let a coworker edit | Share button → add them → **Edit** |
| Let someone only comment | Share button → add them → **Comment** |
| Tidy finished tasks | **Archive Done** |
| Save | Nothing — it's automatic |
